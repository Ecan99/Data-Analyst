# Data-Analyst
This is my personal portfolio website showcasing my journey as a Data Analyst. The site highlights my skills in SQL, Python, Tableau, Power BI, and Excel, and provides a dedicated space to feature my analytics projects.

Features:
 - Responsive design with modern UI/UX.
 - Hero section introducing my profile and career focus.
 - About section with background and education.
 - Skills section highlighting technical expertise.
 - Projects section showcasing dashboards and data analysis work (Tableau, Excel, MySQL).
 - Services section outlining what I offer (data cleaning, dashboard development, SQL querying).
 - Contact section with links to GitHub, LinkedIn, and email.
